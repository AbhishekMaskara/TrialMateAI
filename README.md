# TrialMate AI

## AI-Powered Clinical Trial Site Copilot

### Overview

TrialMate AI is an AI-powered Clinical Trial Site Copilot designed to support newly onboarded Clinical Research Coordinators (CRCs).

The platform leverages Google's Gemini AI model and Streamlit to simplify key clinical trial workflows including protocol interpretation, patient eligibility screening, clinical research knowledge support, and investigator communication.

---

## Problem Statement

Clinical Research Coordinators often face challenges during onboarding due to:

* Complex clinical trial protocols
* Regulatory and GCP requirements
* Patient eligibility assessment
* Investigator communication and documentation

TrialMate AI aims to reduce this learning curve and improve site-level productivity.

---

## Features

### 1. Clinical Research Knowledge Assistant

Provides CRC-focused explanations of clinical research concepts such as:

* GCP (Good Clinical Practice)
* SAE (Serious Adverse Event)
* SDTM
* Clinical trial terminology

---

### 2. Protocol Interpreter

Converts protocol text into:

* Plain language summaries
* Inclusion criteria
* Exclusion criteria
* CRC action points

---

### 3. Patient Eligibility Screener

Assists with preliminary participant screening based on protocol-defined criteria.

Example:

* Age
* Diagnosis
* HbA1c
* Pregnancy status
* Renal disease status

---

### 4. Investigator Brief Generator

Creates concise investigator-ready summaries including:

* Patient overview
* Eligibility assessment
* Key findings
* Recommended next steps

To run TrialMate AI:

1. Create a Gemini API key from Google AI Studio.
2. Open gemini_config.py
3. Replace PASTE_YOUR_API_KEY_HERE with your API key.
4. Run:
   streamlit run app.py
